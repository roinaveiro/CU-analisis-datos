Condicionales y bucles

Las condicionales y los bucles son las dos estructuras de programaci√≥n m√°s importantes cualquiera sea el lenguajes que utilicemos.

  1. Las condicionales permiten ejecutar cierto c√≥digo en funci√≥n de si se cumple una condici√≥n l√≥gica.

  2. Los bucles permiten repetir cierto c√≥digo un n√∫mero espec√≠f√≠co de veces.

1. Condicionales

Mandan a R realizar una tarea si se cumple una condici√≥n

  haz esto
}

Los if reciben una expresi√≥n l√≥gica (TRUE o FALSE).

Pueden anidarse

  if(x>=5){
    print("N√∫mero en [5,8]")
  }
}
## [1] "N√∫mero en [5,8]"

Haz lo anterior con un √∫nico if.

1. Condicionales

if se complementa con else

  Sino haz esto otro

  haz esto
